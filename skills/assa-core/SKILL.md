# ASSA Core Evolution

This skill enables the agent to evolve its operational guidelines based on project-specific patterns and user feedback.

## Features
- **Automatic Pattern Distillation**: Synchronously processes realizations before each turn.
- **Global Wisdom Promotion**: Elevates mature patterns to the universal library.
- **Time-Travel Resilience**: Heals memory state during conversation rewinds.

## Usage
The evolution process is mostly automatic through hooks. When you realize a new technical pattern or user preference, use the `submit_memory_signal` tool provided by the `assa-mcp` server.
