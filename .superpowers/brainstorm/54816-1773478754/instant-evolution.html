The "Hot-Reload" Advantage:

  • Feedback-Driven: Negative feedback (e.g., "Don't use semicolons") triggers an immediate update to user_habits.md.
  • Git-Aware: A git rollback is treated as a critical failure signal, forcing the agent to analyze and avoid that pattern instantly.
  • Zero Latency: The Context Router re-scans memory before every message, ensuring the agent "gets smarter" mid-conversation.
