Key Distillation Stages:

Delta Capture

Implicitly learns your Habits by analyzing git diff between my code and your final version.

Abstraction Filter

Converts surface-level fixes into Low-Level Patterns (e.g., "Always use composition over inheritance").

Promotion Logic

Moves validated local patterns into the Global LIBRARY/ after multi-project verification.
